# [Project 1: Buying Behaviour of Nigerians](https://github.com/BioZen1/Buying-Behaviour-of-Nigerians)

This is a collaborated project i did with other data analyst during the training, where we did an analysis on how Nigerians buy and consume things 

* Data was collected through a secondary model; Google form
* The model uses manual filtering and collaborative filtering techniques
* The analysis was based on 107 respondents in total
* Focused on customers consuming habits
* All age distribution was targetted
* Respondents age were predominantly young adults, between the age of 18 and 35
* Location was within Nigeria
